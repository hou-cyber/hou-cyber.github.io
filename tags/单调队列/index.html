带限制的子序列和

 2021-11-29 2021-11-29 每日一题 / 动态规划4 minutes read (About 660 words)

题目

描述

​ 给你一个整数数组 nums 和一个整数 k ，请你返回非空子序列元素和的最大值，子序列需要满足：子序列中每两个 相邻 的整数 nums[i] 和 nums[j] ，它们在原数组中的下标 i 和~满足 i < j 且 j - i <= k 。

示例

  • 示例1：

  • 示例2：

  • 示例3：

提示

  • 1 <= k <= nums.length <= 10^5
  • -10^4 <= nums[i] <= 10^4

思路

  • 类型：动态规划 + 单调队列优化

  • 实现：循环、双向队列

  • 分析：求子序列和的最大值，比较容易想到的是动态规划。建立数组dp，dp[i]表示以第i个元素结尾的子序列的最大值（注意：该子序列中要包含第i个元素）。状态转移方程为：
    $$
    dp[i]=\max (dp[j])+nums[i]，其中 \max(i-k, 0) \leq j<i
    $$
    需要注意的是只用动态规划会超时，既要进行优化。看状态转移方程发现，对于dp[i]每次要寻找前k个dp[j]的最大值，时间复杂度为k。因此，该题难点在于如何优化该寻找时间，这其实又是求滑动窗口最大值问题。

代码 c++

  • 动态规划：会超时。时间复杂度：O(nk)

  • 动态规划 + 单调队列优化 时间复杂度：O(n)
